# Paradise Nursery Shopping Application

## GitHub Repository

https://github.com/sudha-sambath/e-plantShopping

## Tech Stack

- React
- Redux Toolkit
- React Router
- Vite
- JavaScript (ES6+)
- CSS3

## About Project

Paradise Nursery is an e-commerce web application for browsing and purchasing house plants. Users can explore different categories of plants, view product details, add items to their shopping cart, adjust quantities, and review the total cost before checkout.

The application is built with React, Redux Toolkit, and Vite, providing a responsive and interactive shopping experience.

## Key Features

- Browse plants by category
- Add products to cart
- Increase or decrease quantities
- Remove products from cart
- Real-time cart total calculation
- State management using Redux Toolkit
- Responsive user interface

## Installation

```bash
git clone https://github.com/sudha-sambath/e-plantShopping
cd e-plantShopping
npm install
```

## Usage

Start the development server:

```bash
npm run dev
```

Open your browser and visit:

http://localhost:5173

To preview the production build:

```bash
npm run build
npm run preview
```

Then visit:

http://localhost:4173
